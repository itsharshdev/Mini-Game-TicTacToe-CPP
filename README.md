# Tic Tac Toe Game

A console-based Tic Tac Toe game developed in C++.

## Features

* Two-player gameplay
* Dynamic board updates after every move
* Win detection
* Draw detection
* Replay option
* Uses loops, arrays, and conditional statements

## Technologies Used

* C++
* Arrays
* Loops
* Conditional Statements

## Project Description

This project demonstrates fundamental programming concepts through an interactive Tic Tac Toe game. Players take turns placing their marks on the board, and the system automatically detects wins, draws, and allows replaying the game.
